# ScoreRankForTdmx
| Mono | IL2CPP |
|:--:|:--:|
|✅|❌|

 Adds Score Rank icons to TDMX.
 
 They function exactly the same as they do on Nijiro, where when you hit certain score milestones in a song, you will get the next higher score rank.
